  <action>
    Create `openmob_hub/lib/services/device_manager.dart` with class `DeviceManager`:

    **Constructor:** Takes `AdbService _adb` parameter.

    **State (rxdart):**
    - `final _devices = BehaviorSubject<List<Device>>.seeded([]);`
    - `ValueStream<List<Device>> get devices$ => _devices.stream;`
    - `List<Device> get currentDevices => _devices.value;`

    **refreshDevices() method:**
    1. Call `_adb.listRawDevices()` to get serial+status pairs
    2. For each device with status == 'device' (connected):
       - Call `_enrichDevice(serial)` to get full metadata
       - Determine `connectionType`: if serial starts with 'emulator-' -> 'emulator', if serial contains ':' -> 'wifi', else -> 'usb'
    3. Update `_devices.add(enrichedList)`

    **_enrichDevice(String serial) method:**
    Run these ADB commands in parallel via `Future.wait`:
    - `['shell', 'getprop', 'ro.product.model']`
    - `['shell', 'getprop', 'ro.product.manufacturer']`
    - `['shell', 'getprop', 'ro.build.version.release']`
    - `['shell', 'getprop', 'ro.build.version.sdk']`
    - `['shell', 'wm', 'size']`
    - `['shell', 'dumpsys', 'battery']`
    Parse each result:
    - Screen size: regex `(\d+)x(\d+)` on wm size output
    - Battery level: regex `level: (\d+)` from dumpsys battery
    - Battery status: regex `status: (\d+)` (2=charging, 3=discharging, 5=full)
    Return a fully populated `Device` object.

    **connectWifi(String ipPort) method:**
    1. Run `_adb.runGlobal(['connect', ipPort])`
    2. Check stdout contains 'connected' or 'already connected'
    3. If success, call `refreshDevices()` and return true
    4. Else return false

    **enableWifiAdb(String usbSerial, {int port = 5555}) method:**
    1. Run `_adb.run(usbSerial, ['tcpip', '$port'])`
    2. Wait 2 seconds
    3. Get IP via `_adb.run(usbSerial, ['shell', 'ip', 'route', 'show', 'dev', 'wlan0'])`
    4. Extract IP with regex `src (\d+\.\d+\.\d+\.\d+)`
    5. Call `connectWifi('$ip:$port')`

    **getDevice(String id) method:** Return device from currentDevices where device.id == id, or null.

    **startBridge(String deviceId) / stopBridge(String deviceId):**
    - Update device status to 'bridged' / 'connected' and bridgeActive to true/false via copyWith
    - Emit updated list to _devices

    **dispose() method:** Close _devices BehaviorSubject.

    IMPORTANT: No setState anywhere. All state flows through BehaviorSubject.

  <action>
    1. Create `openmob_hub/lib/services/screenshot_service.dart` with class `ScreenshotService`:

       **Constructor:** Takes `AdbService _adb`.

       **captureScreenshot(String serial) method:**
       - Use `_adb.runBinary(serial, ['exec-out', 'screencap', '-p'])` -- MUST use exec-out, NOT shell (avoids PTY binary corruption per research pitfall #1)
       - Convert bytes to base64 string: `base64Encode(bytes)`
       - Return `({String base64, int width, int height})` record
       - Get width/height from device metadata or parse PNG header (first 24 bytes contain IHDR with width at offset 16 and height at offset 20, both 4 bytes big-endian)
       - If PNG parsing fails, return width=0, height=0

    2. Create `openmob_hub/lib/services/ui_tree_service.dart` with class `UiTreeService`:

       **Constructor:** Takes `AdbService _adb`.

       **getUiTree(String serial, {UiTreeFilter? filter}) method:**
       1. Run `_adb.run(serial, ['exec-out', 'uiautomator', 'dump', '/dev/tty'])` -- use exec-out for stdout
       2. Strip the trailing "UI hierchary dumped to: /dev/tty" message from output using `replaceAll`
       3. Parse XML using `XmlDocument.parse(xml)` from the `xml` package
       4. Iterate `doc.descendants.whereType<XmlElement>()`, skip elements where `name.local != 'node'`
       5. For each node element, extract attributes: `text`, `class`, `resource-id`, `content-desc`, `bounds`, `visible-to-user`
       6. Parse bounds string `[left,top][right,bottom]` using regex `\[(\d+),(\d+)\]\[(\d+),(\d+)\]` into a `Rect`
       7. Assign sequential `index` (starting at 0) to each node
       8. If `filter` is provided, only include nodes where `filter.matches(node)` returns true. But IMPORTANT: index assignment happens BEFORE filtering so indices are stable regardless of filter.
       9. Return `List<UiNode>`

       **Import:** `import 'package:xml/xml.dart';`

       Handle errors gracefully:
       - If uiautomator dump fails (animated screen), return empty list with no exception
       - If XML parsing fails, return empty list
       - Wrap in try/catch and return `[]` on any exception
  </action>
