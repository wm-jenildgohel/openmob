POST /api/v1/devices/:id/bridge/stop      -> {"status":"stopped"}

  <action>
    Create `openmob_hub/lib/services/action_service.dart` with class `ActionService`:

    **Constructor:** Takes `AdbService _adb` and `UiTreeService _uiTree`.

    **All methods return `Future<ActionResult>` and wrap in try/catch returning ActionResult.fail on error:**

    1. **tap(String serial, int x, int y):**
       `_adb.run(serial, ['shell', 'input', 'tap', '$x', '$y'])`
       Return `ActionResult.ok()`

    2. **tapElement(String serial, int index):**
       - Get UI tree via `_uiTree.getUiTree(serial)`
       - Find node where `node.index == index`
       - If not found, return `ActionResult.fail('Element with index $index not found')`
       - Compute center: `node.bounds.centerX`, `node.bounds.centerY`
       - Call `tap(serial, centerX, centerY)`

    3. **typeText(String serial, String text):**
       - Escape special characters: replace space with `%s`, escape `&`, `<`, `>`, `'`, `"`, `(`, `)`, `;`, `|`, backtick with backslash prefix
       - `_adb.run(serial, ['shell', 'input', 'text', escaped])`
       - Return `ActionResult.ok()`

    4. **swipe(String serial, int x1, int y1, int x2, int y2, {int durationMs = 300}):**
       `_adb.run(serial, ['shell', 'input', 'swipe', '$x1', '$y1', '$x2', '$y2', '$durationMs'])`

    5. **pressKey(String serial, int keyCode):**
       `_adb.run(serial, ['shell', 'input', 'keyevent', '$keyCode'])`

    6. **goHome(String serial):**
       `pressKey(serial, AdbKeyCodes.home)`

    7. **launchApp(String serial, String packageName):**
       `_adb.run(serial, ['shell', 'monkey', '-p', packageName, '-c', 'android.intent.category.LAUNCHER', '1'])`

    8. **terminateApp(String serial, String packageName):**
       `_adb.run(serial, ['shell', 'am', 'force-stop', packageName])`

    9. **openUrl(String serial, String url):**
       `_adb.run(serial, ['shell', 'am', 'start', '-a', 'android.intent.action.VIEW', '-d', url])`

    10. **longPress(String serial, int x, int y, {int durationMs = 1500}):**
        `_adb.run(serial, ['shell', 'input', 'swipe', '$x', '$y', '$x', '$y', '$durationMs'])`

    11. **pinch(String serial, int centerX, int centerY, {bool zoomIn = true, int distance = 200, int durationMs = 500}):**
        For zoom in: two swipes from outer points toward center (run both, don't await individually -- best effort)
        For zoom out: two swipes from center toward outer points
        This is approximate since ADB doesn't natively support multi-touch well. Return ActionResult.ok() with a note.

    12. **gesture(String serial, String type, Map<String, dynamic> params):**
        Switch on type:
        - `'long_press'` -> call longPress with params x, y, duration
        - `'pinch_in'` -> call pinch with zoomIn=true
        - `'pinch_out'` -> call pinch with zoomIn=false
        - `'double_tap'` -> tap twice with 100ms delay
        - default -> ActionResult.fail('Unknown gesture type: $type')
  </action>

    1. Create `openmob_hub/lib/server/routes/health_routes.dart`:
       - Function `Router healthRoutes()` returning a Router
       - `GET /health` -> returns `Response.ok(jsonEncode({"status": "ok"}))`

    2. Create `openmob_hub/lib/server/routes/device_routes.dart`:
       - Function `Router deviceRoutes(DeviceManager dm, ScreenshotService ss, UiTreeService uts)` returning a Router
       - All handlers parse JSON body via `jsonDecode(await request.readAsString())` for POST requests
       - All handlers return `Response.ok(jsonEncode(...))` or `Response.notFound(jsonEncode({"error":"Device not found"}))` or `Response.internalServerError(body: jsonEncode({"error": message}))`

       Endpoints:
       - `GET /` -> `dm.currentDevices.map((d) => d.toJson()).toList()` as JSON array
       - `GET /<id>` -> `dm.getDevice(id)?.toJson()` or 404
       - `GET /<id>/screenshot` -> call `ss.captureScreenshot(device.serial)`, return `{"screenshot": base64, "width": w, "height": h}`
       - `GET /<id>/ui-tree` -> parse query params `text` and `visible`, create UiTreeFilter if provided, call `uts.getUiTree(device.serial, filter: filter)`, return `{"nodes": nodes.map((n) => n.toJson()).toList()}`
       - `POST /connect-wifi` -> parse `{"address": "IP:PORT"}`, call `dm.connectWifi(address)`, return result
       - `POST /<id>/bridge/start` -> `dm.startBridge(id)`, return `{"status": "active"}`
       - `POST /<id>/bridge/stop` -> `dm.stopBridge(id)`, return `{"status": "stopped"}`

       Helper: `_getDeviceOrNull(DeviceManager dm, String id)` that returns device or null.

    3. Create `openmob_hub/lib/server/routes/action_routes.dart`:
       - Function `Router actionRoutes(ActionService action, DeviceManager dm)` returning a Router
       - All POST handlers: parse JSON body, validate device exists, call action method, return ActionResult.toJson()

       Endpoints (all under `/<id>/` prefix):
       - `POST /<id>/tap` -> if body has `index`, call `action.tapElement(serial, index)` else call `action.tap(serial, x, y)`
       - `POST /<id>/swipe` -> `action.swipe(serial, x1, y1, x2, y2, durationMs: duration ?? 300)`
       - `POST /<id>/type` -> `action.typeText(serial, text)`
       - `POST /<id>/keyevent` -> `action.pressKey(serial, keyCode)`
       - `POST /<id>/launch` -> `action.launchApp(serial, package)`
       - `POST /<id>/terminate` -> `action.terminateApp(serial, package)`
       - `POST /<id>/open-url` -> `action.openUrl(serial, url)`
       - `POST /<id>/gesture` -> `action.gesture(serial, type, params)`

    4. Update `openmob_hub/lib/server/api_server.dart`:
       - Constructor now takes `DeviceManager`, `ScreenshotService`, `UiTreeService`, `ActionService` as parameters
       - Mount routes: `_router.mount('/health', healthRoutes().call)`
       - Mount device routes at `/api/v1/devices` using `deviceRoutes(dm, ss, uts)`
       - Mount action routes at `/api/v1/devices` using `actionRoutes(action, dm)`
       - Note: shelf_router handles overlapping mount points -- device routes handle GET endpoints and action routes handle POST endpoints under the same prefix

    5. Update `openmob_hub/lib/main.dart`:
       - Create all service instances: `adbService`, `deviceManager = DeviceManager(adbService)`, `screenshotService = ScreenshotService(adbService)`, `uiTreeService = UiTreeService(adbService)`, `actionService = ActionService(adbService, uiTreeService)`
       - Pass all services to `ApiServer(deviceManager, screenshotService, uiTreeService, actionService)`
       - Call `deviceManager.refreshDevices()` after server starts
       - Make all service instances accessible (top-level late final variables) so UI can use them later

    6. Run `dart analyze lib/` to verify.
