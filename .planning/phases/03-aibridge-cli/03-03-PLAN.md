         - Register routes using Go 1.22+ method routing:
           - mux.HandleFunc("GET /health", h.HandleHealth)
           - mux.HandleFunc("GET /status", h.HandleStatus)
           - mux.HandleFunc("POST /inject", h.HandleInject)
           - mux.HandleFunc("DELETE /queue", h.HandleQueueClear)
         - Wrap mux with CORS middleware (allows localhost origins, Content-Type header, all methods)
         - Create http.Server with Addr, Handler, ReadTimeout: 30s, WriteTimeout: 30s
       - func (s *Server) Start() error:
         - Log "HTTP server listening on {addr}"
         - Call s.srv.ListenAndServe()
         - Return err (ignore ErrServerClosed)
       - func (s *Server) Shutdown(ctx context.Context) error:
         - Call s.srv.Shutdown(ctx)
       - func corsMiddleware(next http.Handler) http.Handler:
         - Returns HandlerFunc that sets:
           - Access-Control-Allow-Origin: * (localhost tool, safe)
           - Access-Control-Allow-Methods: GET, POST, DELETE, OPTIONS
           - Access-Control-Allow-Headers: Content-Type
         - If OPTIONS request: return 204
         - Otherwise: call next.ServeHTTP

    2. Create internal/server/handlers.go:
       - Package server
       - Import bridge package, encoding/json, net/http, context, time
       - type Handlers struct:
         - bridge *bridge.Bridge
         - timeout time.Duration
       - Helper: func writeJSON(w http.ResponseWriter, status int, data any):
         - Set Content-Type: application/json
         - Set status code
         - json.NewEncoder(w).Encode(data)
       - Helper: func writeError(w http.ResponseWriter, status int, msg string):
         - writeJSON with {"error": msg}

       - func (h *Handlers) HandleHealth(w http.ResponseWriter, r *http.Request):
         - writeJSON(200, {"status": "ok", "version": "0.1.0"})

       - func (h *Handlers) HandleStatus(w http.ResponseWriter, r *http.Request):
         - writeJSON(200, {
             "idle": h.bridge.IsIdle(),
             "queue_length": h.bridge.Queue().Len(),
             "child_running": h.bridge.IsChildRunning(),
             "uptime_seconds": int(h.bridge.Uptime().Seconds()),
           })

       - func (h *Handlers) HandleInject(w http.ResponseWriter, r *http.Request):
         - Check bridge.IsChildRunning() -- if not: 503 "child process not running"
         - Decode JSON body into InjectRequest{Text string, Priority bool}
         - Validate Text not empty -- 400 "text is required"
         - Check sync mode: r.URL.Query().Get("sync") == "true"
         - If sync:
           - EnqueueWithChan(text, priority)
           - If ErrQueueFull: 429 "queue full"
           - Call bridge.NotifyEnqueue()
           - context.WithTimeout(r.Context(), h.timeout)
           - select on SyncChan or ctx.Done()
           - If timeout: 408 "injection timeout"
           - If success: writeJSON(200, {id, queued: false, position: 0})
         - If async:
           - Enqueue(text, priority)
           - If ErrQueueFull: 429 "queue full"
           - Call bridge.NotifyEnqueue()
           - writeJSON(200, {id, queued: true, position: queue.Len()})

       - func (h *Handlers) HandleQueueClear(w http.ResponseWriter, r *http.Request):
         - count := h.bridge.Queue().Clear()
         - writeJSON(200, {"cleared": count})

       - Request/Response types (unexported or exported):
         - type InjectRequest struct { Text string `json:"text"`; Priority bool `json:"priority,omitempty"` }
         - type InjectResponse struct { ID string `json:"id"`; Queued bool `json:"queued"`; Position int `json:"position"` }
         - type HealthResponse struct { Status string `json:"status"`; Version string `json:"version"` }
         - type StatusResponse struct { Idle bool `json:"idle"`; QueueLength int `json:"queue_length"`; ChildRunning bool `json:"child_running"`; UptimeSeconds int `json:"uptime_seconds"` }
         - type QueueClearResponse struct { Cleared int `json:"cleared"` }
         - type ErrorResponse struct { Error string `json:"error"` }
