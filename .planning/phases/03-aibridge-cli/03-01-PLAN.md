    3. Create internal/bridge/ansi.go:

       - func StripANSI(s string) string -- uses ansiRegex.ReplaceAllString

    4. Create internal/patterns/patterns.go:
       - Package patterns
       - type Pattern struct with Regex string field
       - var BuiltinPatterns map[string]Pattern with entries:
         - "claude": {Regex: `thinking`}
         - "codex": {Regex: `esc to interrupt`}
         - "gemini": {Regex: `esc to cancel`}
       - var DefaultPattern = Pattern{Regex: `esc to interrupt`}
       - func GetPattern(toolName string) *Pattern -- looks up in BuiltinPatterns, returns nil if not found

    5. Create internal/bridge/pty.go (build tag: !windows):
       - Package bridge
       - type PTY struct with fields: cmd *exec.Cmd, ptmx *os.File, oldState *term.State, paranoid bool, injectDelayMs int, writeMu sync.Mutex, closed atomic.Bool, onOutput func(line string)
       - func NewPTY(cmd *exec.Cmd, paranoid bool, injectDelayMs int, onOutput func(string)) *PTY
       - func (p *PTY) Start(ctx context.Context) error:
         - Uses pty.Start(p.cmd) to start command in PTY
         - Handles SIGWINCH for window resize (signal.Notify + goroutine + pty.InheritSize)
         - Sends initial SIGWINCH
         - Sets terminal raw mode via term.MakeRaw(int(os.Stdin.Fd()))
         - Stores oldState for cleanup
         - Starts readLoop goroutine
         - Starts stdin forward goroutine (io.Copy with writeMu protection)
       - func (p *PTY) readLoop(ctx context.Context):
         - 4KB read buffer
         - Reads from ptmx, writes to os.Stdout
         - Calls StripANSI on output, splits by newline, calls p.onOutput for each line
         - Checks p.closed on read error to exit gracefully
       - func (p *PTY) InjectText(text string) error:
         - Acquires writeMu
         - Writes text bytes to ptmx
         - If paranoid, returns (no Enter)
         - If injectDelayMs > 0, sleeps
         - Writes "\r" (Enter) to ptmx
       - func (p *PTY) Close() error:
         - Sets closed flag
         - Restores terminal state via term.Restore
         - Closes ptmx
         - Kills child process (SIGTERM, then SIGKILL after 5s)
       - func (p *PTY) Wait() error -- wraps p.cmd.Wait()
       - Use SysProcAttr with Setpgid: true for process group management

    6. Create internal/bridge/pty_windows.go (build tag: windows):
       - Package bridge
       - Stub implementation that imports UserExistsError/conpty
       - Same PTY struct interface but uses conpty.Start instead of creack/pty
       - Mark with TODO comments for full Windows implementation
       - Must compile on Windows even if not fully tested
