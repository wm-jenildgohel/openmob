     - `Future<void> launchApp(String udid, String bundleId) async` -- runs `xcrun simctl launch $udid $bundleId`. Throws on failure.
     - `Future<void> terminateApp(String udid, String bundleId) async` -- runs `xcrun simctl terminate $udid $bundleId`. Throws on failure.
     - `Future<void> openUrl(String udid, String url) async` -- runs `xcrun simctl openurl $udid $url`. Throws on failure.
     - `Future<void> bootSimulator(String udid) async` -- runs `xcrun simctl boot $udid`. Throws on failure.

Each element mapped to UiNode: `AXLabel` -> text, `AXFrame.x/y/width/height` -> Rect (convert frame to bounds: left=x, top=y, right=x+width, bottom=y+height), `role` -> className, `AXUniqueId` -> resourceId, `AXLabel` -> contentDesc. Assign sequential index starting from 0. Parse defensively -- all fields optional with defaults.

     - `Future<void> swipe(String udid, int x1, int y1, int x2, int y2, {int durationMs = 300}) async` -- runs `idb ui swipe $x1 $y1 $x2 $y2 --duration $durationMs --udid $udid`. NOTE: Check if idb supports --duration flag; if not, omit it.
     - `Future<void> typeText(String udid, String text) async` -- runs `idb ui text "$text" --udid $udid`.
     - `Future<void> pressButton(String udid, String button) async` -- runs `idb ui button $button --udid $udid`. Supported buttons: HOME, LOCK, SIDE_BUTTON, SIRI, APPLE_PAY.
